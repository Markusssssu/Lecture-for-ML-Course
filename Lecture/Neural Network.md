# Нейронные сети

Нейронная сеть — модель машинного обучения, вдохновленная работой мозга.

## Нейрон

Нейрон — обобщение линейной функции.

### Этапы работы нейрона

1. Прием сигналов
2. Взвешивание
3. Суммирование
4. Смещение
5. Функция активации
6. Выход

## Функции активации

Sigmoid — диапазон 0..1  
Tanh — диапазон -1..1  
ReLU — обнуляет отрицательные значения
