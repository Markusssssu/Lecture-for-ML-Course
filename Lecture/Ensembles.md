# Ансамблевые методы

Ансамбли — объединение нескольких моделей для повышения точности.

## Бэггинг

Каждая модель обучается на случайной подвыборке данных.

Пример:

Random Forest.

## Бустинг

Каждая новая модель обучается на ошибках предыдущей.

## Стэкинг

Предсказания моделей используются как признаки для метамодели.

## Блэндинг

Предсказания моделей усредняются с фиксированными весами.
