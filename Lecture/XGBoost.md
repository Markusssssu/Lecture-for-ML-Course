# XGBoost

XGBoost — алгоритм градиентного бустинга на деревьях.

## Особенности

### Регуляризация

- L1 (Lasso)
- L2 (Ridge)

### Работа с пропусками

Алгоритм автоматически определяет ветку для пропущенных значений.

### Параллелизация

Поиск лучшего разбиения выполняется параллельно.

## Плюсы

- высокая точность
- высокая скорость

## Минусы

- сложная настройка
- риск переобучения
