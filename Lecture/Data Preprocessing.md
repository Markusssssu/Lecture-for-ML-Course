# Предобработка данных

Предобработка данных — процесс подготовки данных к обучению модели.

## Типы данных

### Числовые
Данные, над которыми имеют смысл арифметические операции.

### Порядковые
Данные, имеющие порядок.

### Категориальные
Данные, принимающие конечное число значений.

---

## Этапы предобработки

1. Работа с пропущенными значениями
2. Кодирование данных
3. Масштабирование
4. Нормализация
5. Работа с выбросами
6. Создание новых признаков
7. Аугментация данных
