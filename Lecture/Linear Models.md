# Линейные модели

Линейные модели предполагают линейную зависимость между признаками и целевой переменной.

## Линейная регрессия

Модель для задач регрессии.

Цель — минимизация ошибки предсказания.

## Логистическая регрессия

Модель для задач классификации.

Использует сигмоидную функцию для перевода линейной комбинации признаков в вероятность.
