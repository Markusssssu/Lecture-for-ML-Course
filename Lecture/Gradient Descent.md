# Градиентный спуск

Градиентный спуск — алгоритм оптимизации, который минимизирует функцию потерь.

## Алгоритм обучения

1. Инициализация параметров
2. Прямой проход
3. Вычисление ошибки
4. Обратное распространение
5. Обновление параметров

## Learning Rate

Параметр, определяющий размер шага обновления весов.
